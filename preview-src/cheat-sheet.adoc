== Read query 


=== Read Query Structure

[.neo4j-ee]
====
[source, cypher, role=noheader]
----
[USE]
[MATCH [WHERE]]
[OPTIONAL MATCH [WHERE]]
[WITH [ORDER BY] [SKIP] [LIMIT] [WHERE]]
RETURN [ORDER BY] [SKIP] [LIMIT]
----

[.description]
Displayed for neo4j-ee
====

====
[source, cypher, role=noheader]

RETURN [ORDER BY] [SKIP] [LIMIT]
----

[.description]
Displayed for neo4j-ee and deprecated
